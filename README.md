# Notes
This is simple note app which store your basic notes.

<ul>
    <li>create note</li>
    <li>read note</li>
    <li>update note</li>
    <li>delete note</li>
</ul>

<H2>Prerequisites</H2>
<hr>
<ul>
    <li>NodeJs</li>
    <li>ReactJs</li>
    <li>MongoDB</li>
    <li>ExpressJs</li>
    <li>Bootstrap</li>
</ul>
